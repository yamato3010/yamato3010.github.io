### 静的ページ

TiddlyWikiは単一HTMLファイルで構成されているため，Googleの検索結果には，Wikiページごとに表示されません。そのため，各ページを分割HTML化し，静的ページとサイトマップを自動作成する作業の流れを自動化したシェルスクリプトを作成しました。

https://github.com/yamato3010/TiddlyWikiBuilder
